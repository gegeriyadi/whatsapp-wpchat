# WhatsApp WP Chat

A simple click to chat button WhatsApp plugin.

### Description

This plugin lets you add a click to chat button at the bottom of your website. A visitor who clicked the button will be directed to a page and redirect to WhatsApp application in his gadget or on WhatsApp web if the visitor opens WhatsApp via desktop.

### Installation

**Manual Installation**

1. Download the plugin as a [ZIP](https://gegeriyadi.com/download/2077/)
2. In your WordPress admin click _Plugin -> Add New -> Upload Plugin._
3. Upload the ZIP file.
4. Activate the plugin.

**Install With Git**

1. Go to your web plugin directory

```
cd /wp-content/plugins
```

2. clone this repo

```
git clone https://github.com/gegeriyadi/whatsapp-wpchat
```

3. Activate the plugin from wp admin.
